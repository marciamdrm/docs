# Introduction

TaskFlow is a simple task management API that lets you create projects, organize 
tasks within them, track progress, and delete tasks when they're done.

## What you can do with TaskFlow

- **Create projects** to group related work together
- **Add tasks** to any project with a title and description
- **Track progress** by updating task statuses (`pending`, `in_progress`, `done`)
- **Delete tasks** that are no longer needed

## Base URL

```bash
https://test-writechoice.onrender.com
```

<Note>
  The API may take up to 60 seconds to respond on the first request if it has
  been idle. This is expected behavior.
</Note>
